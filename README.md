# CETESB Fumaça Preta — PWA Funcional V1.23

## Correções principais

- Removida a rolagem automática para o topo ao clicar/trocar de tela.
- Relatórios/PDF agora abrem primeiro em tela interna de visualização.
- O app não chama mais impressão automaticamente, reduzindo travamentos no celular e computador.
- Botão “Imprimir / salvar PDF” fica manual dentro da tela de relatório.
- Botão para abrir e-mail para o técnico cadastrado via mailto.

## Observação sobre e-mail

O PWA abre o app de e-mail com destinatário, assunto e corpo preenchidos. Por limitação do mailto/navegador, o PDF não é anexado automaticamente. Se necessário, salve o PDF e anexe manualmente.
