# CETESB Fumaça Preta — PWA Funcional V1.24

## Implementado
- Áudio individual para placa, cor, marca e modelo.
- Município da infração preenchido automaticamente quando o Nominatim retornar cidade/município.
- Número do município preenchido automaticamente com base no PDF enviado.
- Base de números municipais embutida no app com 644 municípios extraídos do PDF.
- Foto compactada antes de salvar e botão para remover foto.
- Tela/instrução para instalação no Android e iPhone.
- Perfis locais por técnico.
- Perfil “Agência Ambiental de Paulínia - CJU” visualiza os registros finalizados locais.

## Limitação
Os registros continuam locais no navegador/dispositivo. Para técnicos em celulares diferentes verem os mesmos registros, será necessário backend/banco central em versão futura.
