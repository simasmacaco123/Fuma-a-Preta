## V1.24.3
- Implementada proteção para o botão físico/gesto de voltar do celular.
- Na tela inicial/perfil, o app exige dois acionamentos de voltar em até 2,5 segundos para sair.
- Durante preenchimento de ocorrência, se houver dados digitados ou foto, o primeiro voltar apenas alerta; o segundo volta para a tela inicial sem salvar.
- Essa proteção reduz risco de fechar o app ou perder preenchimento por toque acidental.
