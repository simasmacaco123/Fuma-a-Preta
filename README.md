# CETESB Fumaça Preta — PWA Funcional V1.24.4

## Correções
- O botão “Imprimir / salvar PDF” agora imprime a própria tela de relatório, sem abrir nova aba intermediária.
- A impressão oculta cabeçalho, botões e demais telas, imprimindo apenas o relatório.
- O botão “Enviar e-mail ao técnico cadastrado” na tela de relatório usa o texto combinado de consulta de placa.
- O relatório consolidado continua em tela interna e pode ser impresso/salvo como PDF pelo mesmo botão.
- Mantém proteção de voltar duas vezes para sair e demais melhorias da V1.24.
